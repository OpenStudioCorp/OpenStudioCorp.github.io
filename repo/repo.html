welcome to the official PythonicOS repository

here you can find all the packages you need to install

to install a package, type pycake install [package] in the shell

to uninstall a package, type pycake uninstall [package] in the shell

to update a package, type pycake update [package] in the shell

to update all packages, type pycake update in the shell

to list all packages, type pycake list in the shell

to search for a package, type pycake search [package] in the shell

to get help, type pycake help in the shell

to get help for a command, type help [command] in the shell
